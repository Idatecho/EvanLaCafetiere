Pour ce projet, nous utiliserons L’IDE pycharm.

PyCharm est un environnement de développement intégré (IDE) pour Python, développé par JetBrains. Il offre une interface conviviale pour écrire, organiser et exécuter du code Python.

Voici les éléments clés de l'interface de PyCharm :

  1. Fenêtre de projet : Située à gauche, elle affiche les fichiers et dossiers de votre projet.
  1. Éditeur de code : Au centre, c'est ici que vous écrivez votre code.
  1. Barre d'outils : En haut, elle contient des boutons pour exécuter, déboguer, etc.
  1. Console : En bas, elle affiche la sortie de votre programme.
  1. Barre d'état : En bas de la fenêtre, elle indique des informations sur le projet et l'IDE.

🧰 Fonctionnalités utiles de PyCharm :

  • Complétion de code : PyCharm suggère automatiquement des complétions de code pour accélérer la programmation.
  • Débogueur intégré : Permet de mettre des points d'arrêt et d'exécuter le code pas à pas.
  • Gestion des environnements virtuels : Facilite la création et la gestion d'environnements Python isolés.
  • Intégration avec Git : Offre des outils pour le contrôle de version directement dans l'IDE.

Les classes :

Les classes en Python permettent de créer ses propres types d'objets, avec des attributs (des variables liées à l'objet) et des méthodes (des fonctions qui agissent sur l'objet). C’est un outil central de la programmation orientée objet (POO).

Exemple :

class Player:
    def __init__(self, nom, nb_vie, nb_degat): #pose les variables qu'on va utiliser
        self.nom = nom
        self.nb_vie = nb_vie
        self.nb_degat = nb_degat # par la suite, on appelera self."lavariable"

    def attaquer(self, autre_joueur): # on creer une fonction "attaquer"
        print(f"{self.nom} attaque {autre_joueur.nom} et inflige {self.nb_degat} dégâts.")
        autre_joueur.nb_vie -= self.nb_degat
        if autre_joueur.nb_vie <= 0:
            autre_joueur.nb_vie = 0
            print(f"{autre_joueur.nom} est mort.")
        else:
            print(f"{autre_joueur.nom} a encore {autre_joueur.nb_vie} points de vie.")

# Création des joueurs
joueur1 = Player("Evan", 100, 15)
joueur2 = Player("Robin", 80, 20)

# Combat
joueur1.attaquer(joueur2)
joueur2.attaquer(joueur1)
joueur1.attaquer(joueur2)
